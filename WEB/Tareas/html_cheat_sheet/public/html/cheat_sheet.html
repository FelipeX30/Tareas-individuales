Cheat Sheet de HTML

a - Hyperlinks

Se utiliza para crear un enlace a otra página web.

<a href="https://www.google.com.mx">Visita google.com.mx</a>

p - Paragraph

Define un párrafo de texto.

<p>Este es un ejemplo de párrafo en HTML.</p>

h1, h2, h3, h4, h5, h6 - Headings

Representan seis niveles de encabezados en HTML, siendo h1 el más importante y h6 el menos importante.

<h1> Encabezado Nivel 1</h1> <h2> Encabezado Nivel 2 </h2> <h3> Encabezado Nivel 3</h3> <h4> Encabezado Nivel 4</h4> <h5>Encabezado Nivel 5</h5> <h6>Encabezado Nivel 6</h6>

div - Division

Se utiliza para agrupar contenido y/o estilos en un contenedor.

<div>Este es un contenedor div.</div>

span - Span

Se usa para agrupar elementos en línea dentro de un documento.

<span>Este texto está dentro de un span.</span>

strong - Strong Importance

Indica que el texto es de gran importancia.

<strong>Texto importante</strong>

i - Italic

Se usa para texto en itálica.

<i>Texto en itálica</i>

u - Underline

Subraya el texto.

<u>Texto subrayado</u>

b - Bold

Se utiliza para dar énfasis al texto, mostrándolo en negrita.

<b>Texto en negrita</b>

strike - Strikethrough

Representa un texto sobre el cual se ha pasado una línea, indicando que está eliminado o incorrecto.

<strike>Texto tachado</strike>

small - Small Text

Se utiliza para indicar texto menos importante.

<small>Texto pequeño</small>

em - Emphasis

Enfatiza el texto y que generalmente se muestra en itálica.

<em>Texto enfatizado</em>

header - Header Element

Se utiliza para contener elementos introductorios.

<header>Aquí va el encabezado de la página</header>

section - Section

Define una sección en un documento, como capítulos o encabezados.

<section>Aquí va una sección del documento</section>

article - Article

Representa una composición autónoma en un documento, página o aplicación, que es destinada a ser distribuida de forma independiente.

<article>Aquí va un artículo</article>

aside - Aside

Se utiliza para contenido tangencialmente relacionado con el contenido alrededor del cual se coloca.

<aside>Información adicional aquí</aside>

footer - Footer

Define el pie de página de un documento o sección.

<footer>Este es el pie de página</footer>

nav - Navigation

Se usa para contener la navegación principal de un documento.

<nav><a href="#home">Inicio</a>

img - Image

Se utiliza para incorporar una imagen en el documento.

<img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcQCQCgBGuRFnXZQYrXCHVGS8lm0bwwPKwbBED6Ejwwsow&s" width="500px" height="400px"> <img src="../images/ferrari-f8-tributo-2019-05-1400x637.webp">

br - Line Break

Inserta un salto de línea en el texto.

Primera línea<br>Segunda línea

hr - Horizontal Rule

Representa un cambio temático entre párrafos.

<hr>

form - Form

Se utiliza para crear un formulario para la entrada del usuario.

<form><label>Nombre:</label><input type="text" name="nombre"></form>

table - Table

Se utiliza para mostrar datos en forma de tabla.

<table> Encabezado 1 Encabezado 2 Dato 1 Dato 2 </table>

ul - Unordered List

Define una lista desordenada.

<ul> Elemento 1 Elemento 2 </ul>

ol - Ordered List

Define una lista ordenada.

<ol> Primer elemento Segundo elemento </ol>
